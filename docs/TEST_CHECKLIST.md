# Test Checklist

Manual test checklist for **Yelkenli Yaşam Tycoon**.

## Build

- [ ] Run `npm run build`
- [ ] Confirm the build completes without errors

## Mobile UI

- [ ] Check the game in a mobile browser layout
- [ ] Confirm text remains readable on small screens
- [ ] Confirm buttons and main actions are comfortable for touch
- [ ] Confirm important player status remains visible

## Core Gameplay

- [ ] Produce content successfully
- [ ] Confirm content cooldown starts and displays correctly
- [ ] Start a route successfully
- [ ] Trigger and review a sea decision event
- [ ] Complete a route successfully
- [ ] Review the arrival reward screen for clarity and correct rewards

## Upgrades

- [ ] Start an upgrade successfully
- [ ] Confirm the upgrade timer runs and updates correctly

## Daily Goals

- [ ] Confirm daily goals progress updates correctly
- [ ] Confirm 3/3 completion gives the +2,500 TL bonus
- [ ] Confirm daily goals state remains correct after save/load

## Progression

- [ ] Confirm captain XP increases correctly
- [ ] Confirm level-up bonus is awarded correctly
- [ ] Confirm the global progress strip updates and displays correctly

## Persistence

- [ ] Save the game and reload it
- [ ] Confirm content cooldown persists correctly
- [ ] Confirm route-related state persists correctly
- [ ] Confirm upgrades and timers persist correctly
- [ ] Confirm daily goals progress persists correctly
- [ ] Confirm captain XP and level state persist correctly

## Android

- [ ] Run Android debug build when relevant
- [ ] Confirm the latest changes do not break the Android workflow
