# Yelkenli Yaşam Tycoon Agent Kuralları

## Çalışma Kuralı
- Bir toplu işi aynı anda çalıştırın.
- Aktif görev dışında gitmekten kaçının.
- Git durumu(dirty) olduğunda başlamadan önce durdurun.
- Kayıt/yükleme, otomatik kaydetme, rotalar, deniz modu, yükseltme zamanlayıcı, sponsor ekonomisi veya temel para/follower ekonomisine dokunmayın (belirli bir izin verilmedikçe).
- Paket yüklemesi yapmayın (belirli bir izin verilmedikçe).
- Otomatik olarak göndermeyin.
- Kod değişiklikleri yapıldığında `npm run build` çalıştırın.
- Bir hata 5 kez çözülmezse, `errors_log.md`'ye kaydedin ve sonraki göreve geçin.
- Her günün sonunda `progress.md`'yi güncellerin.

## Güvenlik Kuralları
- Sadece izin verilen dosyaları değiştirin.
- Yalnızca izin verilen kod değişikliklerini yapın.
- Yalnızca izin verilen hata düzeltmelerini yapın.
